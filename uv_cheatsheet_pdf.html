🚀 uv コマンド チートシート

超高速 Python パッケージマネージャ - Rust製で10-100倍高速

クイックスタート

# インストール確認 uv version # 新規プロジェクト uv init myproject && cd myproject # パッケージ追加 + 実行 uv add numpy pandas uv run python main.py

プロジェクト管理

uv init <name>新規プロジェクト作成
uv sync依存関係インストール
uv lockロックファイル更新
uv tree依存関係ツリー表示

仮想環境

uv venv仮想環境作成
uv venv --python 3.11指定バージョンで作成
source .venv/bin/activate有効化 (Linux/Mac)
.\.venv\Scripts\Activate.ps1有効化 (Windows)

パッケージ管理

uv add <package>パッケージ追加
uv add <pkg>==1.0.0バージョン指定
uv add --dev pytest開発依存として追加
uv remove <package>パッケージ削除

uv run (推奨)

# スクリプト実行 uv run python script.py uv run main.py # ツール実行 uv run pytest tests/ uv run ruff check . # 一時的な依存関係 uv run --with requests python -c "import requests"

Python バージョン

uv python install 3.11Python インストール
uv python listインストール済み一覧
uv python pin 3.11バージョン固定

ツール管理

uv tool install ruffグローバルツール
uv tool listツール一覧
uvx <tool>一時実行

uvx (一時実行)

# 試すだけ（インストール不要） uvx cowsay "Hello World" uvx httpie GET https://api.github.com # テストツール uvx pytest uvx --with pytest pytest tests/

pip 互換

uv pip install <pkg>pip install 互換
uv pip listパッケージ一覧
uv pip freezerequirements出力

ワークフロー

新規プロジェクト

uv init myproject && cd myproject uv add fastapi uvicorn uv run uvicorn main:app --reload

既存プロジェクト

cd project && uv sync uv run python main.py

テスト環境

uv add --dev pytest uv run pytest tests/ -v

設定ファイル

pyproject.toml

[project] name = "myproject" dependencies = ["fastapi"] [project.optional-dependencies] dev = ["pytest", "ruff"]

.python-version

3.11

Tips

  • uv run で仮想環境自動使用
  • uvx で環境を汚さず試行
  • --with で一時依存関係追加
  • Python バージョン自動ダウンロード
  • pip の 10-100倍 高速実行
  • 設定不要でそのまま使用可能
