Pro Tips

  • uv run は仮想環境を自動作成・使用
  • uvx は環境を汚さず一時実行
  • --with で一時的な依存関係追加
  • Python バージョンは必要時に自動ダウンロード
  • pip の 10-100倍 高速実行
  • 設定ファイル不要でそのまま使用可能
  • .python-version ファイルを自動認識
  • Git リポジトリからの直接インストール対応
